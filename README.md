# wzry-voice

> Yunzai-Bot 王者荣耀语音图片插件  
> 发送「王者荣耀」或「王者」，机器人依次回复：**随机图片（引用）→ 固定语音 → 随机语音**

## 📦 安装

在 Bot 目录下执行：

```bash
git clone --depth=1 https://github.com/anyliew/wzry-voice.git ./plugins/wzry-voice
```

重启 Bot 即可。

## 🎮 使用

| 触发词              | 回复内容                                                    |
| :------------------ | :---------------------------------------------------------- |
| `王者荣耀` / `王者` | 随机图片（引用原消息）→ 固定语音 `leige-001.mp3` → 随机语音 |

## 🔄 更新

| 命令            | 权限 | 说明                           |
| :-------------- | :--- | :----------------------------- |
| `#wzry更新`     | 主人 | 普通 `git pull`                |
| `#wzry强制更新` | 主人 | 强制重置至远端（丢弃本地修改） |

## 📁 目录结构

```
wzry-voice/
├── apps/               # 核心逻辑
├── images/wzry/        # 随机图片 (.webp)
└── voice/
    ├── leige-001.mp3   # 固定语音
    └── wzry/           # 随机语音 (.mp3)
```
