
# Wzry Voice

> 王者荣耀语音插件 —— 一句话触发三步连发：固定语音 + 随机语音 + 随机图片

## 简介

本插件专为 [Yunzai-Bot](https://github.com/TimeRainStarSky/Yunzai) 或其衍生框架设计。当用户在群聊/私聊中输入 `王者荣耀` 或 `王者` 时，插件会自动：

1. 发送一条固定的 MP3 语音（`leige-001.mp3`）
2. 从 `voice/wzry/` 文件夹中随机选取一条 MP3 语音发送
3. 从 `images/wzry/` 文件夹中随机选取一张 WebP 图片发送

三步依次执行，带来丰富的交互体验。

## 安装

1. 将插件文件夹放置于 Bot 的 `plugins` 目录下：
```
git clone --depth=1 https://github.com/anyliew/wzry-voice.git ./plugins/wzry-voice
```
2. 确保 Bot 已加载插件（通常重启 Bot 加载命令）。

## 触发词

| 触发词 | 说明 |
|--------|------|
| `王者荣耀` | 完整触发词 |
| `王者`     | 简写触发词 |

> 注意：触发词为 **完全匹配**，前后不能有其他字符。

## 目录结构
```

wzry-voice/
├── wzry-voice.js # 插件主文件
├── voice/
│ ├── leige-001.mp3 # 【必选】第一步固定的语音文件
│ └── wzry/ # 【必选】第二步随机语音文件夹
│ ├── 1.mp3
│ ├── 2.mp3
│ └── ... # 任意数量 .mp3 文件
└── images/
└── wzry/ # 【必选】第三步随机图片文件夹
├── 1.webp
├── 2.webp
└── ... # 任意数量 .webp 文件

```
### 文件要求

- **语音文件**：`.mp3` 格式，建议编码为单声道、16k 采样率，以保证 QQ 语音正常播放。
- **图片文件**：`.webp` 格式，QQ 原生支持，无需转码。
- 文件名无特殊限制，支持中文（但建议使用英文/数字避免编码问题）。

## 配置

本插件无需额外配置文件。所有行为由文件夹内容决定：

- 若 `voice/leige-001.mp3` 缺失，第一步会报错并回复提示，但不会影响后续步骤。
- 若 `voice/wzry/` 下没有 `.mp3` 文件，第二步会跳过并提示。
- 若 `images/wzry/` 下没有 `.webp` 文件，第三步会跳过并提示。

## wzry-voice 插件更新器

一个用于自动更新 `wzry-voice` 插件的机器人指令模块。支持**普通更新**和**强制更新**两种模式，更新完成后会推送详细的变更信息（提交日志、文件列表等）。

## 功能特性

- 🔄 **一键更新**：通过聊天指令拉取 `wzry-voice` 插件的最新代码。
- ⚡ **双模式**：
  - 普通更新：执行 `git pull`，保留本地未提交的修改（如有冲突需手动处理）。
  - 强制更新：执行 `git fetch --all` + `git reset --hard origin/<分支>`，**丢弃所有本地变更**，完全同步远程仓库。
- 📊 **详细反馈**：更新后输出提交记录（最多5条）、变更文件列表（最多10个）、新旧版本哈希、执行耗时等。
- 🧹 **错误诊断**：针对常见 Git 错误（路径不存在、网络失败、认证失败等）给出明确提示。

## 使用方法

### 命令格式

| 命令 | 说明 |
|------|------|
| `#wzry更新` | 普通更新，执行 `git pull` |
| `#wzry强制更新` | 强制更新，丢弃本地修改并重置到远程分支 |

> 命令**不区分大小写**，例如 `#WZRY更新` 或 `#Wzry强制更新` 同样有效。

### 权限要求

仅 **主人（master）** 可用，普通成员发送命令无响应。

## 工作流程

1. 检查插件目录 `./plugins/wzry-voice/` 是否存在。
2. 获取当前 Git 提交哈希（旧版本）。
3. 自动识别当前分支（若处于 detached HEAD 状态则尝试获取远程默认分支）。
4. 执行更新：
   - 普通模式：`git pull`
   - 强制模式：`git fetch --all` + `git reset --hard origin/<分支>`
5. 获取新版本哈希，比较差异。
6. 如有更新，提取变更文件列表和提交日志。
7. 生成汇总报告，并通过**合并转发消息**发送到群聊（若转发失败则直接发送文本）。



## 日志

插件运行时会在控制台输出带有 `[wzry-voice]` 前缀的详细日志，包括：
- 文件是否存在
- 随机选择的文件名
- 发送成功/失败状态

便于排查问题。
